# Chapter 1 — I Thought It Would Be Simple

I wanted to build my own Taiwan stock analysis program.

At first, I believed it would be a small personal project.

I wasn't trying to build a platform.

I just wanted a better tool for myself.

Download stock prices.

Store them in a database.

Run a few technical indicators.

Build some reports.

That was all I thought I needed.

Then I started running into problems that I had never expected.

Some of them seemed surprisingly simple, yet they kept appearing again and again.

What surprised me even more was that I couldn't find anyone discussing these problems in detail online.

I thought downloading stock prices would be the hardest part.

I was wrong.

but this was only the beginning.
