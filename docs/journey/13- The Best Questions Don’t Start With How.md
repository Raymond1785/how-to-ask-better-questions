## 13. The Best Questions Don’t Start With How

When we want to build something, the first question is often:

**“How do we do this?”**

How do we get the data?

How do we make it work?

These are useful questions.

But they can also make us start solving too early.

Sometimes, the better questions are:

**Why does this need to exist?**

**What problem are we actually trying to solve?**

**Who really needs the answer?**

Once those questions become clear, the “how” often becomes much easier.

I learned that a difficult problem does not always need a better answer.

Sometimes, it needs a better question.

**The best questions don’t start with how. They help you discover what the “how” should be.**
