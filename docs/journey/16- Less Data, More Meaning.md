## 16. Less Data, More Meaning

At first, every new piece of data felt valuable.

If we could collect it, we wanted to show it.

But as the system grew, something unexpected happened.

More information did not always make the screen more useful.

Sometimes, it made the important things harder to see.

The question was no longer:

**What else can we show?**

It became:

**What does the user actually need to see?**

Not every number deserves the same attention.

Not every indicator needs to be visible at the same time.

Sometimes, making a system better means removing information instead of adding more.

Because the goal is not to show everything we know.

**The goal is to make what matters easier to see.**
