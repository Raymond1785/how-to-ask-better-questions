# Journey

> Every project leaves behind a lesson.

Building software is rarely a straight line.

Sometimes the biggest breakthrough comes from a mistake.

Sometimes one simple question changes the direction of an entire project.

This series is a collection of lessons learned while building real-world products.

Not theories.

Not motivational quotes.

Just experiences that changed how I think, design, and build.
